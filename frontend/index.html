How to use

  1. Ensure the backend API is running by executing ./start_backend.sh from the project root.
  2. Prepare a CSV file with at least two columns: description and amount.
  3. Optional columns: date, currency, labeled_category (for accuracy scoring).
  4. Click Choose CSV file or drag and drop your file into the upload zone.
  5. Click Categorize to process the file.
  6. Review results in the table. Use category filter buttons to focus on specific groups.
  7. Click Close Results to reset and upload another file.
Example input: description = "Paid $12.50 at Corner Cafe." → Expected tag: Food
